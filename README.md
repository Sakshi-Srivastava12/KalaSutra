# KalaSutra

## Overview
KalaSutra is a full-stack e-commerce platform designed for artists and customers. Artists can upload and manage their artwork, while customers can browse, purchase, and track products online.

## Features
- User Authentication (Customer, Artist, Admin)
- Product Upload and Management
- Shopping Cart
- Order Processing
- Video Showcase
- REST API Integration
- Role-Based Access Control
